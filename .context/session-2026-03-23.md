## Architecture Overview

```
moonshine-daemon (main.go)
├── Transcriber (internal/moonshine/) - CGO bindings to libmoonshine
├── Daemon (internal/daemon/)
│   ├── daemon.go - State machine, text processing pipeline
│   ├── socket.go - Unix socket IPC server
│   └── output.go - wl-copy, wtype, notify-send, pw-play
├── Audio (internal/audio/)
│   ├── recorder.go - Single-shot pw-record
│   ├── stream_recorder.go - Streaming pw-record --raw
│   ├── devices.go - PipeWire device enumeration
│   └── wav.go - WAV parsing, normalization
├── Config (internal/config/) - ~/.config/moonshine/config
└── Tray (internal/tray/) - System tray UI via fyne.io/systray
```
